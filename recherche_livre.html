Recherche de livre

Présentation du projet

Ce projet est un site web développé en duo, ayant pour objectif l’intégration d’une API externe. Il permet à l’utilisateur de rechercher un livre à partir de son titre et d’afficher les informations correspondantes. Ce travail avait pour finalité de mettre en pratique l’exploitation d’une API au sein d’une application web dynamique.

Aperçu de l’interface

Page d’accueil

La page d’accueil représente l’interface principale de l’application. Elle propose une barre de recherche permettant de rechercher des livres par titre, accompagnée de filtres pour affiner les résultats. Elle intègre également une fonctionnalité de favoris, offrant aux utilisateurs la possibilité de sauvegarder et de consulter facilement les livres sélectionnés

Objectifs

Intégration d’une API

Utiliser une API externe pour récupérer des données dynamiques en temps réel.

Recherche de données

Permettre la recherche de livres à partir de leur titre et afficher les résultats associés.

Exploitation des données

Analyser et structurer les informations retournées par l’API pour les rendre lisibles et utiles à l’utilisateur.

Interface utilisateur

Concevoir une interface simple, intuitive et agréable pour faciliter la navigation et la recherche.

Technologies utilisées

Fonctionnalités

Affichage des resultats

Les résultats de recherche sont générés à partir des données récupérées via l’API. Les ouvrages dont le titre se rapproche le plus de la requête de l’utilisateur sont affichés en priorité. Les informations reçues sont ensuite traitées et organisées afin de proposer une présentation claire, cohérente et facile à consulter.

Favoris

L’application intègre une fonctionnalité de gestion des favoris permettant de sauvegarder des livres directement depuis les résultats de recherche. Les ouvrages enregistrés restent accessibles à tout moment via une liste dédiée, facilitant leur consultation ultérieure. Les favoris peuvent être ajoutés ou supprimés aussi bien depuis la page de recherche que depuis la liste des favoris, garantissant une expérience utilisateur fluide et efficace.
